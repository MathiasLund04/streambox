### **Hvordan I greb test drevet arbejde an:**
- Vi startede med at køre hele testprogrammet og se hvor der var fejl
- Derefter tog vi en test af gangen og fik til at virke (fra toppen til bunden og ved brug af Arrange-Act-Assert strukturen)

 
### **Hvad der var sværest:**
- Det sværeste at lave i dette program var testen for mostViewedInGenre_shouldReturnHighestViews_andTieBreakByTitleAsc()
- Det der gjorde det svært for os var at vi skulle have mange betingelser på og kunne sende en optional retur-værdi tilbage fra metodekaldet for at sikre os at det var den rigtige film der blev send tilbage.
